Virtual Office AI・像素風 AI 辦公室

把你手上的 AI 訂閱（Claude Code / OpenAI / Gemini）或 API 金鑰，變成一支可以派任務的像素員工團隊 —— 用既有訂閱就能跑，不用另外付 API 費。手機開網頁就能遠端指揮自己電腦上的 AI：左側名冊管理員工、中間辦公室即時走動、右側即時回收任務成果。

  • Phaser 4 像素辦公室：場景編輯器、MMO 式捏臉合成器、內建新手教學

  • 自帶金鑰（BYO-key）：支援 Anthropic / OpenAI / Gemini / OpenRouter，由瀏覽器直連 AI 服務，預設不儲存、不經伺服器
  • 逐字串流輸出、對話紀錄、檔案拖放上傳下載
  • Google 登入 + Supabase 雲端同步；沙盒權限控管，員工可存取的資料夾由網頁遠端授權
